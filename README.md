# Ibexa Core

Ibexa Core is a successor of Ibexa Kernel (formerly eZ Platform Kernel) and the heart of
Ibexa DXP and Ibexa Open Source.

See [the official documentation](https://doc.ibexa.co/) and learn more about 
[Ibexa DXP](https://www.ibexa.co/products) editions:
* [Ibexa Content](https://github.com/ibexa/content)
* [Ibexa Experience](https://github.com/ibexa/experience)
* [Ibexa Commerce](https://github.com/ibexa/commerce)
